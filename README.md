This project is a recreation of the **[Zentry](https://zentry.com/)** website, a visually captivating and interactive design. Featuring scroll-triggered animations, geometric transitions, and immersive video storytelling, the website aims to deliver a luxurious and modern feel. The project focuses on engaging UI/UX and smooth responsiveness, mimicking the elements that make it a potential Awwwards winner.
